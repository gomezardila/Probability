# Probability-Mathematics

# SOA P Exam – Ejercicios Resueltos en Python y R

Este repositorio contiene una colección de ejercicios resueltos del **Examen P (Probability)** de la **Society of Actuaries (SOA)**, con soluciones desarrolladas en **Python** y **R**. Está orientado como material de estudio para candidatos que se preparan para el examen, combinando teoría probabilística y aplicación computacional.

## Objetivo

El propósito es:
- Reforzar los fundamentos de probabilidad requeridos para el Examen P.
- Aplicar conceptos teóricos mediante simulaciones y cálculos computacionales.
- Practicar ejercicios tipo examen con soluciones claras y reproducibles.

## Contenidos

- Conceptos básicos de probabilidad
- Técnicas de conteo
- Probabilidad condicional e independencia
- Variables aleatorias discretas y continuas
- Distribuciones discretas (Bernoulli, Binomial, Geométrica, Poisson)
- Distribuciones continuas (Uniforme, Exponencial, Normal)
- Funciones de distribución y densidad
- Esperanza, varianza y momentos
- Distribuciones conjuntas
- Covarianza y correlación
- Simulación de variables aleatorias
- Ejercicios tipo examen P con soluciones detalladas

## Tecnologías

- Python 3.x
- R (versión ≥ 4.0)
- Bibliotecas:
  - Python: `numpy`, `scipy`, `pandas`, `matplotlib`
  - R: `stats`, `tidyverse`, `ggplot2`
